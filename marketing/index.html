Make a password in 3 easy steps

Enter a nickname.

  • Examples:
  • Work Email 2019
  • My Gmail
  • Amazon Personal

Enter a master password.

This master password will be used in AgiliKEY to generate all other passwords.
You can finally use that one password you have memorized.

Retype your master password.

If this is your first time generating a password, it is good practice to reconfirm the master password by typing it a second time.
This way the app will generate a password using the correct information.
